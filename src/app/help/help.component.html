Help for searching occurrences

The form supports setting the following parameters (none required):
  1. Selecting a species (bird, mammal, insect ..)
  2. Traverse the taxonomic tree to the left to find it (for now you need to know the latin (scientific) name!)
    Click the header labels till you see a list of species
    shows the total number of occurrences in DOFbasen
  3. Selecting an observer (source and reporter of occurence)
  4. Type an observer's full name (from DOFbasen) in the input text field to the right
  5. Narrowing the area using the map
  6. Draw a polygon by clicking the map - finish by double-clicking.
    You don't have to finish where you started
Launching search and more:
  • Click the search button. Results will show in list. Maximum number of occurences in result is 300
  • The occurence list contains the species name the site name and the number of specimens occuring in the record
    Click the site name and the map will zoom and pan to the site.
    For now the species names show in Danish.
